- Container = div in html, container has child, 스타일링을 위해 decoration 사용 -> BoxDecoration -> 스타일 설정 (ex Color)

- important: container is gonna occupy the exact space of content, it's not gonna stretch by default

- mainAxisAlignment: top to bottom align
- croessAxisAlignment: left to right align (stretch 하면 해당 row 에 해당하는 크기 전부 차지)

- stateful vs stateless widget

- extending in flutter such as StatelessWidget class, which will take on all the properties and abilites
  of StatelessWidget, yet will allow us to add our own properties and abilites

- @override annotation in dart is something that simply provides the compiler a bit more information as how
  to treat a given function or method. So this will 'override' the parent StatelessWidget's build method

- context 는 react context api 의 context 와 같은 개념, flutter 에서 context 를 쓰려면 BuildContext 를 같이 사용해야 한다.

- Widget \_section(String title, Color:color) = Widget is return type, \_section is function name

- private function is a function that cannot be used in out side of class: function with underscore (\_section)

- final = once define, then impossible to redefine

- constructor is a special function or method that let you specify parameter of your instances of class you're creating
- contructor create a instance of class

- model is data, model can have a relation such as Location and LocationFact (use this structure when you use data in your application)

- Image.network(url,fit) for url image

- Container -> constraints -> BoxConstraints -> tightFor / loose / tight
