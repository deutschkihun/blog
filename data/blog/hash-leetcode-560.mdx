## 1.문제 설명

배열 nums 과 integer k 가 주어진다. subarray 의 합이 k 가 되는 모든 subarray 의 갯수를 구하시오. subarray 는 연속되고 빈배열이 아닌 배열을 말한다.

## 2. 문제 접근

subarray 이기 때문에 처음부터 끝까지 iteration 을 돌려봐야 한다. 예를 들어 nums = [1,2,3], k=3 가정하자.
hash 를 만들고 각각의 subarray 의 합을 저장해주어야 한다.

```tsx

hash
sum = 0


iteration 1 : sum += nums[i] = 1
iteration 2 : sum += nums[i] = 3 = k
iteration 3 : sum += nums[i] = 6 != k

```

마지막 iteration 같은 경우 전체합이 6이 된다. 그런데 nums[i] 에 해당하는 값은 3이며 총합 6에서 k를 빼주게 되면 3이된다. 즉 subarray의 각각의 요소의 합을 계속 더해나가되 k 와의 차이를 비교하여 hash 에 존재하게 되면 조건을 충족하게 된다고 이해 하면 된다.

한가지 주의 할 점은 hash 에 초기값으로 key,value 를 0,1로 지정해 주어야 한다. 그렇지 않으면 처음에 한번에 원하는 결과를 얻은 경우를 커버 할 수 없다.

## 3. 풀이

```tsx
var subarraySum = function (nums, k) {
  let count = 0
  let sum = 0
  let hash = new Map()
  hash.set(0, -1)

  for (let i = 0; i < nums.length; i++) {
    sum += nums[i]
    if (hash.has(sum - k)) {
      count += hash.get(sum - k)
    }
    hash.set(sum, hash.get(sum) + 1 || 1)
  }
  return count
}
```
