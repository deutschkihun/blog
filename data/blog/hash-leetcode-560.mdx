## 1.문제 설명

3의 제곱수를 생각하자. 3의 0제곱, 3의 1제곱, 3의 2제곱, ... 은 순서대로 1, 3, 9, 27, ... 이 된다. 이를 바탕으로, 한 개 이상의 서로 다른 3의 제곱수의 합으로 표현되는 수를 생각할 수 있다. 예를 들어 30은 27과 3의 합이므로, 이러한 수에 포함된다. 한 개 이상의 서로 다른 3의 제곱수의 합으로 표현되는 N번째로 작은 수를 구하는 프로그램을 작성하시오

## 2. 문제 접근

숫자와 관련된 문제는 일단 2가지 경우의 수를 고려해보자

- 2진법 (혹은 n진법)
- 몫과 나머지의 연관성

이문제는 2진법으로 입력받은 숫자를 변환하고 변환된 이진수의 1로 표기된 부분을 3의 승으로 취해 숫자를 10진법으로 변환시켜주면 된다.

예를들어

```tsx

n = 4

4 -> 100 -> 3**2  = 9

11 -> 1011 -> 3**3 + 3**1 + 3**0 = 31

```

## 3. 풀이

```tsx
function solution(n) {
  const binaryCode = n.toString(2).split('')
  return binaryCode.reduce((a, b, index) => {
    if (b === '1') a += Math.pow(3, binaryCode.length - 1 - index)
    return a
  }, 0)
}
```
