## 1.문제설명

무한수열 s = (1), (1,2), (1,2,3)... 이있다. 이러한 수열을 군수열이라고 한다.

이때 k 번째 항을 구하시오

## 2.문제접근

예를 들어 50번쨰 항을 구하는 경우라고 가정해보다.

1군까지 = 1개
2군까지 = 1+2개
3군까지 = 1+2+3개
n군까지 = 1+2+3+...+n-1+n개

따라서 n군까지의 총갯수는 n(n+1)/2 이다. 즉 9군은 45개 10군은 55개가 된다.
50번째 항은 9군 + 5번째 10군항 = 5이다.

## 3. 문제 풀이

위의 접근 방식을 코드로 옮기면 다음과 같다.

```tsx
function gun(k) {
  let secondlastGunIndex = 0
  for(let i=0; i<k; i++) {
    const num = i*(i+1)/2
    if(k<=num) {
      secondlastGunIndex = i-1
      break
    }
  }
  return k-secondlastGunIndex(secondlastGunIndex+1)/2
```
