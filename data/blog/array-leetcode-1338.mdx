## 1.문제 설명

integer 배열이 주어진다. 배열안의 숫자중 임의 갯수만큼 integer 를 하나의 set 으로 묶어서 지정한다. set 으로 지정된 임의의 갯수의 숫자를 배열에서 모두 제거했을때 배열의 전체길이가 반이하가 되게끔 하는 최소한의 set 갯수를 구하시오.

## 2. 문제 접근

일단 배열안에 어떠한 숫자가 있는지 그리고 그 숫자가 각각 몇개가 있는지를 파악하는 것이 중요하다. 이럴때 hash 를 쓰는 것이 가장 좋다. 공간복잡도도 낮고 시간복잡도도 낮아서 필자가 가장 추천하는 방법이다.

그리고 나서 갯수가 많은 순으로 오름차순 정렬을 해준다. 이제

따라서 이러한 문제를 방지하기 위해서는 루프를 뒤에서부터 돌려야 한다.

## 3. 풀이

뒤에서 루프를 돌리면 다음과 같이 구현 해볼수 있다.

```tsx
var minSetSize = function (arr) {
  let hash = new Map()
  let target = Math.ceil(arr.length / 2)
  let sum = 0
  let i = 0

  for (let item of arr) {
    hash.set(item, hash.get(item) + 1 || 1)
  }

  const temp = [...hash.entries()].map((e) => e[1]).sort((a, b) => b - a)
  while (sum < target) {
    sum += temp[i]
    count++
    i++
  }
  return count
}
```
