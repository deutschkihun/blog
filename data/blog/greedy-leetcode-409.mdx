## 1.문제 설명

문자열 s 가 주어진다. s 내에서 가장긴 팰린돔 (앞뒤가 똑같은 문자열)을 찾으시오. 대문자와 소문자는 다른 문자로 취급한다.

## 2. 문제 접근

팰린듬의 가장 큰 특징은 1개의 홀수 문자열과 나머지 짝수개의 쌍의 문자열로 구성되어 있다. 따라서 2개의 쌍을 찾을 떄마다 카운팅을 2개씩 늘려주면 된다.

주의

만약 2개씩 찾을 때마다 카운팅을 늘려주는게 아니라 일단 전체 갯수를 모두 확인하고 거기서 짝수개만 다 더해주는 방식을 생각해볼수도 있다. 이거도 사실 가능하긴 하지만 edge case 가 존재한다. 예를들어 `ccc` 라는 문자열이 있다고 가정해보자. 이런경우 갯수를 세보면 c 가 홀수개라서 카운팅 대상에 포함되지 않는다. 하지만 `ccc` 는 팰린돔이 맞기 때문에 전체 갯수를 확인하고 거기서 짝수만 세는 방식은 모든 경우의 수를 다 커버 할 수 없다.

## 3. 풀이

```tsx
var longestPalindrome = function (s) {
  let hash = new Map()
  let max = 0
  for (const str of s) {
    hash.set(str, hash.get(str) + 1 || 1)
    if (hash.get(str) % 2 === 0) max += 2
  }
  return s.length > max ? max + 1 : max
}
```
