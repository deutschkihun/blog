## 1. 슬라이딩 윈도우 알고리즘

슬라이딩 윈도우 알고리즘은 일정한 범위(윈도우) 내에서 이동(Sliding) 하는 것을 말한다. 예를 들어 2가지 unique 문자만 포함한 문자열을 찾는 문제가 있다고 가정했을때 슬라이딩 윈도우 알고리즘을 사용하면 쉽게 문제를 풀수 있다. 또는 배열이 주어지고 배열안의 sub-array 의 합이 max 가 되는 경우를 찾는경우에도 사용 할 수 있다.

## 2. 코드 구현

위에서 설명한 내용을 기반으로 예제 문제를 풀어보겠습니다.

배열이 주어지고 사이즈가 k인 subarray 에서 subarray의 요소합이 가장 큰 결과를 리턴해야 한다.

```tsx
// O(n) solution for finding maximum sum of a subarray of size k
function maxSum(arr, k) {
  let n = arr.length
  let max = 0
  let sum = 0
  // find initial sum of first k elements
  for (let i = 0; i < k; i++) {
    sum += arr[i]
    max = sum
  }

  for (let i = k; i < n; i++) {
    sum += arr[i] - arr[i - k]
    if (sum > max) max = sum
  }
  return max
}

// Driver code
let arr = [1, 4, 2, 10, 2, 3, 1, 0, 20]
let k = 4
console.log(maxSum(arr, k))
```
