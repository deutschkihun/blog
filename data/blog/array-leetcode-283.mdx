## 1.문제 설명

배열안의 0 을 모두 뒤로 보내고 리턴하면 된다. 또한 space complexity 는 O(1) 로 제한한다.

## 2. 문제 접근

이문제는 space complexity 에 대한 제한이 없었다면 정말 쉬운 문제인데, 제한 사항을 줌으로써 생각을 하게 만드는 문제이다. 가장 쉽게 생각 할 수 있는 접근 방법은 루프롤 돌려서 0이 나오면 해당 부분을 잘라서 맨뒤로 넣으면 된다. 다만 이렇게 할 때 생길수 있는 edge case 가 있다.

예를 들어 [0,0,1,2,3] 이라는 배열이 있다고 가정해보자.

- 루프를 돌려서 1번째 iteration 에서 0을 만나게 된다. 해당 0을 splice 한다.
- 그렇게 되면 이제 0번째 자리에는 2번쨰 0이 들어간다.
- 그리고 2번쨰 iteration 이 되면 i=1 이 되기 때문에 맨앞에 있는 0을 놓치게 된다.

따라서 이러한 문제를 방지하기 위해서는 루프를 뒤에서부터 돌려야 한다.

## 3. 풀이

뒤에서 루프를 돌리면 다음과 같이 구현 해볼수 있다.

```tsx
var moveZeroes = function(nums) {
  for(let i=nums.length-1, i>=0; i++) {
    if(nums[i]===0) {
      nums.splice(i,1)
      nums.push(0)
    }
  }
};
```

감사합니다.
